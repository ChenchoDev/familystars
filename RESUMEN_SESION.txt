 🚀 QUÉ HACER AHORA
═══════════════════════════════════════════════════════════════════════════════

  PASO 1: Copia .env y configura credenciales (5 min)
  $ cd apps/backend
  $ cp .env.example .env
  [Edita: DATABASE_URL, JWT_SECRET]

  PASO 2: Instala dependencias (3 min)
  $ npm install

  PASO 3: Ejecuta migrations (1 min)
  $ npm run migrate

  PASO 4: Inicia servidor (2 min)
  $ npm run dev

  ✅ API viva en http://localhost:3000
